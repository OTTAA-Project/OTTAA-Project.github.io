Manual Testing

Sometimes, manual tests are necessary to find different errors that Automatic Testing can not find.


The errors can be:

  • Design errors
  • Function Bugs
  • Other kinds of bugs
  • Testing Setup
  • Onboarding
  • Main Screen
  • Groups Gallery

Set Up the Test

OTTAA Testing
VersionProduct Version
N° TestingTesting number
DateTesting date
DeviceTesting Device
User NameUser name
GenderUser Gender

Remember :

  • Product Version : Fill this text field with the number version for testing.
  • N° Testing : Fill this text field with the number of testing.
  • Date : Fill this text field with the date of testing.
  • Device : Fill this text field with the name of device to testing.
  • User Name : Fill this text field with the user name who test it.
  • Gender : Fill this text field with the user gender :
    • The Gender can be
      1. Female
      2. Male
      3. Binary
      4. Other
